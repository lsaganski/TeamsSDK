Microsoft Teams Mobile Embed SDK

The Microsoft Teams Mobile Embed SDK enables you to add the Microsoft Teams meeting experience directly in your own app. The Embed SDK is easily added to your app and - with just a few lines of code – allows the app to join a Teams meeting directly.

The Embed SDK is available for iOS and Android. By adding the SDK to your app’s project the app can join a meeting through a single line of code. When the meeting is joined, the normal Teams meeting experience will be initiated from within your app. The app’s user will be able to join a Teams meeting as a guest.
  • Easy as 1, 2, 3: Our SDK is built to be as easy as possible. Import the library, initialize the SDK and then add the meeting coordinates via one line of code.
  • Out-of-the-Box Experience: The full Teams meeting experience is available with no extra code needed. All of the UI is ready for you without having to make a single change.
  • Full Guest Experience: App users don’t need an existing Teams account. The SDK can join any existing Teams meeting from any tenant.

How does the Embed SDK work?

Adding the Teams Embed SDK to your app’s project is just three steps:
  • Import the Embed SDK libraries to your mobile app’s project
  • Initialize the SDK with a single API call
    [MSTUIApplication.sharedInstance initialize];
  • Join a meeting with a URL
    [MSTUIApplication.sharedInstance joinMeetingWith:meetingUrl participantName:name error:error];
